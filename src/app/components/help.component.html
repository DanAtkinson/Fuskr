🖼️What is Fuskr?

Fuskr is a browser extension for Chrome and Firefox that creates image galleries from sequences of numbered images. Many websites store their images with names like photo_001.jpg, photo_002.jpg, photo_003.jpg… Fuskr spots these patterns and automatically loads every image in the sequence for you — no clicking through pages one at a time.

Originally created by Dan Atkinson and Jonathon Bolster, Fuskr has grown from a simple right-click tool into a fully featured gallery viewer with dark mode, keyboard navigation, ZIP downloads, video support, and much more.

🔍How It Works

Fuskr works by detecting the number (or letter) sequence in an image URL, then loading each image in that sequence until they stop being found.

Automatic Detection

Paste any image URL that contains a number and click Generate Gallery. Fuskr will start from that image, count upwards, and stop when images are no longer found. No setup required — it just works.

Example — paste this URL and Fuskr does the rest:

https://example.com/gallery/photo_042.jpg

Fuskr will try photo_043.jpg, photo_044.jpg… and stop when images stop loading.

Specifying a Range

You can tell Fuskr exactly which images to load by replacing the number with a range in square brackets: [start-end]. This is useful when you know the range in advance or when automatic detection loads too many images.

URL Pattern What Fuskr Loads
photo[1-10].jpg photo1.jpg through photo10.jpg — ten images
image[001-050].png image001.png through image050.png — zero-padded numbers
chapter[a-z]/cover.jpg Works with letters too — chaptera/ through chapterz/
vol[1-3]/page[01-20].jpg Multiple patterns — all combinations of volumes and pages
Zero-padding: Using [001-010] instead of [1-10] tells Fuskr to keep the leading zeros, producing 001, 002… rather than 1, 2…

🚀Ways to Use Fuskr

Right-click an Image

On any web page, right-click an image that has a number in its URL. You will see a Fusk sub-menu. The +/− option auto-detects the range, whilst + and − let you count explicitly upwards or downwards from the current image.

Paste a URL Directly

Click the Fuskr icon in your browser toolbar to open the gallery page. Paste your image URL into the box and click Generate Gallery.

Use the Address Bar

Click your browser's address bar, type fuskr followed by a space, then paste your image URL and press Enter. Fuskr will open a new gallery tab instantly.

Create from a Text Selection

If you see a URL written as plain text on a page (rather than an image link), select the URL text, right-click, and choose Fusk → Create from selection.

Manual Gallery

Right-click anywhere and choose Fusk → Manual. This opens the gallery page so you can type a URL from scratch — useful when you want to construct a custom range.

Infinite Mode

Once a gallery is open, enable Infinite Mode from the controls bar. Fuskr will keep loading images beyond the original range in both directions, pausing to ask whether to continue when it hits several consecutive broken images.

⌨️Keyboard Shortcuts

Keyboard shortcuts are active when the gallery is open and no text input is focused.

Gallery Navigation

Key Action
→ or ↓ Next image
← or ↑ Previous image
Home Jump to the first image
End Jump to the last image
Enter or Space Open the highlighted image in the full-screen viewer

Image Viewer

These shortcuts work whilst the full-screen image viewer is open:

Key Action
→ or ↓ Next image
← or ↑ Previous image
Home Jump to the first image
End Jump to the last image
Esc Close the viewer
Shortcuts are automatically disabled when a text input, button, or dropdown has keyboard focus, to prevent accidental navigation whilst typing.

🖥️Display Modes

The gallery offers four ways to view images. Switch between them using the controls bar at the top of the gallery.

Mode Description
Fit Page Each image is scaled down so it fits entirely within the browser window — nothing is cropped.
Full Width Images are stretched to fill the full width of the page. Tall images may require scrolling.
Fill Page Images are zoomed to fill the entire screen. Some edges may be cropped on portrait images.
Thumbnails Small previews displayed in a scrollable grid. Click any thumbnail to open the full-screen viewer.

Your preferred default mode can be set in .

🔒Privacy

Fuskr is designed with privacy in mind. Here is a plain-English summary:

  • ✅ No tracking. Fuskr does not collect, transmit, or share any information about you or your browsing activity.
  • ✅ Local storage only. Your settings and gallery history are stored in your browser's extension storage and never leave your device.
  • ✅ No telemetry. No analytics services, crash reporters, or third-party services of any kind are included.
  • ⚙️ Optional diagnostic logging. An opt-in debug logging feature is available in . It is off by default. When enabled, logs are stored locally only. You decide if and when to share a log file when reporting a bug — and you should always review the contents before doing so, as logs may contain image URLs.
