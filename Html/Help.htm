Introduction

Using a tool like Fuskr can be a little daunting, especially if you've never heard of regular expressions until you read this paragraph!

But don't worry. Fuskr is intended to make creating image galleries simple and easy, and for a lot of use cases, we'll do the leg work for you!

Let's say that you have the following url: http://www.example.com/images/1.jpg. When you right click the image, then you can simply choose the +/-, + or - option.
