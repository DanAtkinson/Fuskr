Foreground

Default option is "enabled".

When you create a new gallery, you may not wish to open it in the foreground. By unchecking this option, any new gallery will appear in the background instead.

History

Default option is "enabled".

Every time you create a new gallery, the extension records the created url (up to the last 10). Clicking on the url will re-generate that gallery.

Last fusks. Fuskr only displays the last 10. Click to revisit.

About Fuskr

Fuskr was borne of a frustation that my favourite Firefox plugin Firefusk created by Jonathan Soma wasn't available for Chrome.

Once I'd stopped crying, I remembered that I was a developer and could do almost anything if I put my mind to it.

Then, I remembered that I was actually a pretty rubbish developer and asked for help from Jonathon Bolster.

Jonathon gratefully helped and actually rewrote pretty much all of the code for Fuskr to make it so damned extensible and also provided lots of unit tests for it.
